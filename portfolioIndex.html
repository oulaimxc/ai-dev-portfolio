Challenge

AI Solution

  • Automated Testing: ໃຫ້ AI ສ້າງ Unit Tests ຄອບຄຸມ Edge cases ຕ່າງໆຢ່າງໄວວາ.
Impact

// Before: Legacy Code (Manual)

function calculate(d) {

let res = [];

for(let i=0; i<d.length; i++) {

if(d[i].active) res.push(d[i].val * 2);

}

return res;

}

// After: Refactored with AI (Clean Code)

const calculateActiveValues = (data) => {

return data

.filter(item => item.active)

.map(item => item.val * 2);

};

Level 3: Expert Prompting Techniques

1 Contextual & Persona Prompting

ກຳນົດບົດບາດໃຫ້ AI ເຊັ່ນ "Act as a Senior System Architect..." ເພື່ອໃຫ້ໄດ້ຜົນລັບທີ່ກົງກັບໜ້າວຽກ.

2 Chain-of-Thought (CoT)

ສັ່ງໃຫ້ AI ຄິດເປັນຂັ້ນຕອນ (Step-by-step reasoning) ເພື່ອແກ້ໄຂ Bug ທີ່ຊັບຊ້ອນ ຫຼື ອອກແບບ Database Schema.

3 Few-Shot Prompting

ໃຫ້ຕົວຢ່າງ (Examples) ການຂຽນ Code ຫຼື Format ຂອງ Document ກ່ອນໃຫ້ AI ສ້າງວຽກຈິງ ເພື່ອຄວບຄຸມ Output ໃຫ້ໄດ້ມາດຕະຖານທີມ.

4 Iterative Refinement

ການດັດປັບ Prompt ຢ່າງຕໍ່ເນື່ອງເພື່ອເພີ່ມປະສິດທິພາບຂອງຜົນລັບ (Zeroing in on the solution).
